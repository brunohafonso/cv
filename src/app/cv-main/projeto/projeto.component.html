BikeMobi

Aplicativo que auxilia na navegação transmitindo o passo a passo da rota para um dispositivo que fica preso a uma luva e que orienta o ciclista durante o trajeto sem a necessidade de olhar o celular, diminuindo assim a chance de roubos e além disso um sistema de avaliação do trajeto para que os trajetos passados fossem cada vez melhores de acordo com o feedback de cada usuário.

Envolvidos

Bruno Afonso, Rebecca Arantes, Rodolfo Andrade, Odirlei Sabella, Emerson Rocha, Murilo Rosa, Kauan Macedo

Técnologias Utilizadas

C#, Asp.Net Core, Entity Framework Core, Sql Server, Swagger,Html, Css, Javascript, React, Android, Iot
