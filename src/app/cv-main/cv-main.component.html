Design Thinking

Encontrar a melhor solução para um desafio ou lançar um novo produto no mercado que realmente tenha valor para o cliente final pode ser muito complicado, mas pra facilitar o processo utilizo o design thinking evitar os erros mais comuns e validar uma idéia antes mesmo de colocar a mão na massa.

Desenvolvimento Back End

Nessa parte que a mágica dos bastidores acontece, todas as funcionalidades do sistema são criadas e configuradas para que se possa funcionar o cadastro de informações, login de usuários e etc, pra isso conto com algumas ferramentas como C#, Asp.Net, Entity Framework, Visual Studio e muitas outras.

Modelagem de Dados

De que adianta ter um sistema web ou um site com um visual super maneiro se quando desligamos o computador as informações são todas perdidas ? pra isso temos o banco de dados onde armazenamos tudo que for relevante para a aplicação e eu busco organizar tudo da melhor forma para que esses dados fiquem seguros e não interfiram na performance.

Metodologias Ágeis

Usando Scrum e Kanban podemos produzir o dobro das coisas em metade do tempo, e claro que nos meus projetos eu uso essas metodologias, fazendo tudo rápido como um foguete mas sem perder a qualidade é claro.

Controle de versão

Já pensou em 5 versões de um mesmo site ou sistema ? como controlar essas versões de forma fácil e simples ? utilizando git e github/bitbucket, assim controlamos a versão e temos um relatório completo de todas as alterações, podendo revertê-las caso necessário.
